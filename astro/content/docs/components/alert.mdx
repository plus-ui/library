The alert component provides contextual feedback messages for typical user actions. It supports various styles, states and customization options to effectively communicate different types of information.

## Anatomy

The alert component consists of several key elements:
- Status icon (optional)
- Message content
- Description (optional) 
- Dismiss button (optional)

<Image src={anatomy} alt="anatomy of alert component" />

## Usage

## API Reference

### Properties

| Property     | Description               | Type     | Values                                     | Default  |
|-------------|---------------------------|----------|-------------------------------------------|----------|
| kind        | Visual style              | string   | 'filled', 'outlined', 'dashed'            | 'filled' |
| status      | Alert type                | string   | 'default', 'success', 'warning', 'danger', 'info' | 'default' |
| size        | Component size            | string   | 'sm', 'md', 'lg'                          | 'md'     |
| message     | Main alert text           | string   | -                                         | ''       |
| description | Additional details        | string   | -                                         | ''       |
| invert      | Inverted color scheme     | boolean  | true, false                               | false    |
| dismissible | Show close button         | boolean  | true, false                               | true     |
| statusIcon  | Custom status icon        | string   | -                                         | ''       |
| dismissIcon | Custom dismiss icon       | string   | -                                         | ''       |
| fullWidth   | Full width display        | boolean  | true, false                               | false    |

### Events

| Event Name    | Description            | Detail                |
|--------------|------------------------|----------------------|
| plus-dismiss | Fired when dismissed   | `{ status: string }` |
