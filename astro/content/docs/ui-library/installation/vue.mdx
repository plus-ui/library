## Vue Installation

Please find below installation guide for integrating the Plus UI component library into a Vue project using Vite:

#### **1. Install Plus UI**

```bash
npm install @plusui/core
```

#### **2. Configure Vite:**

```js
import { defineConfig } from "vite";
import vue from "@vitejs/plugin-vue";

export default defineConfig({
  plugins: [
    vue({
      template: {
        compilerOptions: {
          isCustomElement: tag => tag.startsWith('plus-')
        }
      }
    })
  ]
});
```

<Aside>
To learn more, please check how to use both [Vue and Web Components](https://vuejs.org/guide/extras/web-components).
</Aside>

#### **3. Initialize Components**

```js
import { createApp } from 'vue';
import '@plusui/core/dist/style.css';

const app = createApp(App);
app.mount('#app');
```

#### **4. Use Components:**

```vue
<template>
  <plus-button status="primary" @plus-click="handleClick">
    Click me
  </plus-button>
</template>

<script setup>
const handleClick = () => {
  console.log('Button clicked!');
};
</script>
```
