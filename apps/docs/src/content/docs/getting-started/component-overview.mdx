## Utility Components

Beyond the 29 primary components, Plus UI includes **15 specialized utility components** that enhance functionality:

- **Group Components**: `accordion-group`, `button-group`, `checkbox-group`, `radio-group`, `tab-group`
- **Item Components**: `breadcrumb-item`, `dropdown-item`, `list-box-item`, `segmented-picker-item`, `select-item`, `tab-panel`
- **Container Components**: `toast-container`
- **Foundation Components**: `base`, `caption`, `icon`, `label`, `service`

These utilities work seamlessly with primary components to provide advanced functionality and customization options.
