Plus UI is an open-source project with the goal of helping everyone build better-experienced products. We welcome contributions of all kinds, and there are many ways to get involved!

## Community & Code Contributions

Your expertise can help make Plus UI better for everyone.

*   **Spread the Word**: Share Plus UI with designers and developers you know.
*   **Join the Community**: Help other members on [GitHub Discussions](https://github.com/plus-ui/library/discussions), Discord, and the Figma Community.
*   **Improve Documentation**: Notice a typo or a missing guide? Submit a pull request to our documentation.
*   **Report Bugs**: Find a bug? Please check if it's already reported, and if not, [open a new issue](https://github.com/plus-ui/library/issues/new/choose).
*   **Contribute Code**: We welcome pull requests! Fix a bug or add a new feature.
*   **Request a Feature**: Have a great idea? Share it on our [feature request board](https://plusuitest.canny.io/feature-requests).

## Getting Support

Need help? Here's how to get it.

*   **GitHub Issues**: For bugs and feature requests, this is the best place. [Open an issue](https://github.com/plus-ui/library/issues/new/choose).
*   **Community Channels**: Ask for help and share ideas on our community platforms.
*   **Pro Support**: If you have a Pro Plan, you can get prioritized support by emailing us at [support@plusui.com](mailto:support@plusui.com) with your Order ID.
