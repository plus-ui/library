The Button component enables users to perform actions and make choices with a single tap. It's a fundamental interactive element that can render as either a `<button>` for actions or an `<a>` for navigation when an `href` is provided.

## Accessibility (a11y)

- **Keyboard Behavior:**
  - Buttons and links are focusable via the `Tab` key.
  - Can be activated using `Enter` or `Space` keys when focused.
  - Links support `Enter` key activation, buttons support both `Enter` and `Space`.
- **Screen Reader:**
  - When no `href` is provided, uses the native `<button>` element with implicit `role="button"`.
  - When `href` is provided, uses the native `<a>` element with appropriate link semantics.
  - The text content provided within the component serves as the accessible name.
  - For icon-only buttons/links, provide a descriptive `aria-label` attribute.
- **Required Developer Actions:**
  - Use `href` for navigation, omit it for actions.
  - Ensure button/link text clearly indicates the action or destination.
  - For icon-only components, **always** provide an `aria-label` attribute (e.g., `<plus-button href="/settings" aria-label="Settings"><i class="fas fa-cog"></i></plus-button>`).
  - Ensure sufficient color contrast between the text/icon and background.
  - Use `external` prop for external links to automatically add secure attributes.

## API Reference

### Properties

| Name        | Type                                                                     | Default     | Description                                              | Required |
| ----------- | ------------------------------------------------------------------------ | ----------- | -------------------------------------------------------- | -------- |
| `kind`      | `'filled' \| 'outlined' \| 'dashed' \| 'text'`                           | `'filled'`  | Visual style of the button.                              | No       |
| `status`    | `'default' \| 'primary' \| 'success' \| 'warning' \| 'danger' \| 'info'` | `'default'` | Status/color variant of the button.                      | No       |
| `size`      | `'sm' \| 'md' \| 'lg'`                                                   | `'md'`      | Size of the button.                                      | No       |
| `disabled`  | `boolean`                                                                | `false`     | Disables button interaction.                             | No       |
| `loading`   | `boolean`                                                                | `false`     | Shows loading spinner and disables interaction.          | No       |
| `fullWidth` | `boolean`                                                                | `false`     | Makes the button span the full width.                    | No       |
| `href`      | `string`                                                                 | `undefined` | When provided, renders as anchor link instead of button. | No       |
| `target`    | `string`                                                                 | `undefined` | Specifies where to display the linked URL.               | No       |
| `rel`       | `string`                                                                 | `undefined` | Relationship between current and linked document.        | No       |
| `download`  | `string`                                                                 | `undefined` | Sets link to download instead of navigate.               | No       |
| `external`  | `boolean`                                                                | `false`     | Automatically sets secure attributes for external links. | No       |
| `type`      | `'button' \| 'submit' \| 'reset'`                                        | `'button'`  | Button type for form interactions (button only).         | No       |

### Events

| Name         | Payload Type        | Description                           |
| ------------ | ------------------- | ------------------------------------- |
| `plus-click` | `CustomEvent<void>` | Fired when the button is clicked.     |
| `plus-focus` | `CustomEvent<void>` | Fired when the button receives focus. |
| `plus-blur`  | `CustomEvent<void>` | Fired when the button loses focus.    |

### Slots

| Name                | Description                                     |
| ------------------- | ----------------------------------------------- |
| `default` (unnamed) | The main content of the button (text, etc.).    |
| `prefix`            | Content placed before the default slot content. |
| `suffix`            | Content placed after the default slot content.  |

### CSS Custom Properties

| Name             | Description                            |
| ---------------- | -------------------------------------- |
| `--text-color`   | Controls the text color.               |
| `--border-color` | Controls the border color.             |
| `--bg-default`   | Controls the default background color. |
| `--bg-hovered`   | Controls the background on hover.      |
| `--bg-pressed`   | Controls the background when pressed.  |
| `--bg-focused`   | Controls the background when focused.  |
| `--bg-loading`   | Controls the background when loading.  |

### CSS Shadow Parts

| Part     | Description                |
| -------- | -------------------------- |
| `button` | The main button container. |
