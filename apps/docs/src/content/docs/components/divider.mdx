The Divider component is used to create a visual separation between sections of content or groups of elements. It can be oriented horizontally or vertically and supports different line styles and thicknesses.

## Anatomy

<Image src={anatomy} alt="Divider - anatomy" />

## Usage

### Basic Example

A standard horizontal divider.

<CodePreview autoGeneration={true}>
  <div>
    Content above
    <plus-divider></plus-divider>
    Content below
  </div>
</CodePreview>

### Orientation

Use the `orientation` prop to set the divider direction. Vertical dividers require a defined height on the divider or its container.

<CodePreview autoGeneration={true}>
  <div>
    <plus-divider></plus-divider>
    
    <br />
    
    <div style="height: 50px; display: flex; align-items: center; gap: 1rem;">
      Left
      <plus-divider orientation="vertical"></plus-divider>
      Right
    </div>
  </div>
</CodePreview>

### Kind

The `kind` prop changes the line style.

<CodePreview autoGeneration={true}>
  <div>
    Solid (Default)
    <plus-divider kind="solid"></plus-divider>
    Dashed
    <plus-divider kind="dashed"></plus-divider>
    Dotted
    <plus-divider kind="dotted"></plus-divider>
  </div>
</CodePreview>

### Thickness

The `thickness` prop adjusts the line thickness.

<CodePreview autoGeneration={true}>
  <div>
    Thin (Default)
    <plus-divider thickness="thin"></plus-divider>
    Medium
    <plus-divider thickness="medium"></plus-divider>
    Thick
    <plus-divider thickness="thick"></plus-divider>
  </div>
</CodePreview>

### With Text Content

Place text or other elements in the default slot to display content within the divider line.

<CodePreview autoGeneration={true}>
  
    <plus-divider>Text</plus-divider>
   
</CodePreview>

## Accessibility (a11y)

*   **Screen Reader:**
    *   The component uses `role="separator"` implicitly via the host element or explicitly depending on context, indicating its purpose as a separator.
    *   It is typically non-interactive and not included in the tab order.
    *   If the divider contains text content (via the slot), ensure the text itself is meaningful or provides context.
*   **Required Developer Actions:**
    *   Use dividers semantically to separate distinct groups of content. Avoid using them purely for decoration if a simple margin or padding would suffice.
    *   Ensure sufficient visual contrast between the divider line and the background, especially when customizing colors using `--divider-color`.

## API Reference

### Properties

| Name              | Type                                         | Default        | Description                                                      | Required |
|-------------------|----------------------------------------------|----------------|------------------------------------------------------------------|----------|
| `orientation`     | `'horizontal' \| 'vertical'`                 | `'horizontal'` | Sets the orientation of the divider.                             | No       |
| `kind`            | `'solid' \| 'dashed' \| 'dotted'`               | `'solid'`      | Sets the line style of the divider.                              | No       |
| `thickness`       | `'thin' \| 'medium' \| 'thick'`                | `'thin'`       | Sets the thickness of the divider line.                          | No       |

### Slots

| Name              | Description                                         |
|-------------------|-----------------------------------------------------|
| `default` (unnamed) | Optional content to display within the divider line. |

### CSS Shadow Parts

| Part      | Description                                                       |
|-----------|-------------------------------------------------------------------| 
| `divider` | The main container element (`div`).                               |
| `label`   | The container for the slotted content (when content is present). |

<Aside type="note">
<p>The `label` part is implicitly the container for the slot content but might not be explicitly targeted in all internal styles.</p>
</Aside>

### CSS Custom Properties

| Name                  | Description                                |
|-----------------------|--------------------------------------------|
| `--divider-color`     | Controls the color of the divider line.    |
| `--divider-thickness` | Controls the thickness of the divider line. |
| `--divider-spacing`   | Controls the spacing around the divider.   |
