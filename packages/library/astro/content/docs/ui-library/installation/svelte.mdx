## Svelte Installation

1. Install <mark>plusui core library:</mark>

```bash
npm install @plusui/core
```

2. Import into your project:

```svelte
<script>
  import '@plusui/core/dist/style.css';
</script>

<main>
  <h1>Hello Plus UI</h1>
  <div>
    <plus-button status="primary">Click me</plus-button>
  </div>
</main>

<style>
  /* your styles here */
</style>
```

3. npm run dev&#x20;

## Creating a Svelte project

To begin creating a project with Svelte and Plus UI, follow these steps:

**Set up a Svelte project if you haven't already.** Start by installing Svelte via your terminal:

```bash
npx degit sveltejs/template @plusui/template/svelte-plus-ui
cd svelte-plus-ui
npm install
```

for more information, visit [link](https://svelte.dev/blog/svelte-for-new-developers#creating-a-project).
